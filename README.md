# Student Performance Analysis System

This project analyzes student academic performance using Python and CSV data.
It calculates average marks, identifies top and weak performers, and visualizes results.

## Features
- Reads student data from CSV file
- Calculates total and average marks
- Identifies top and weak performers
- Displays performance using bar chart

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Project Files
- students.csv – dataset
- student_analysis.ipynb – main code
- output_sample.txt – sample output
- performance_chart.png – result visualization

## How to Run
1. Open student_analysis.ipynb in Google Colab
2. Upload students.csv
3. Run all cells
